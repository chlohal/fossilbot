Fossilbot

Fossilbot is a fully-customizable server utility Discord bot.

Get Fossilbot Fossilbot Features

  • Channel Organization

    Enable Channel Organization on any server category to automatically sort your most active channels to the top!

  • Anti-Spam

    Fossilbot can block anything; naughty words, ping-spamming, or mass pinging.

  • Nametag Colors

    Allow users to pick from a list of nametag colors-- or let them choose any color they want! Colors can be restricted to rewards roles.

  • Notification System

    Users can use the notify command to add themselves to a Notification Role, which can be pinged to gather a group!

  • Hamburger Watermelon

    Equal number of features in the list. This is a selling point.

  • Voice Channel Emojis

    Know at a glance what game your friends are playing with automatic emoji labels!

  • Utility Commands

    Grab an image from Google Images, look up a game on Steam, or translate messages into another language!

  • Fully Custom

    Don't like a feature? Just turn it off in the Fossilbot Server Settings!
